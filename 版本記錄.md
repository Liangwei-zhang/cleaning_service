# 版本記錄

## v1.0 - 2026-02-25

### 文件結構
- `index.html` - 首頁（登錄）
- `host.html` - 房東頁面
- `cleaner.html` - 接單員頁面  
- `admin.html` - 管理員頁面
- `api/server.py` - API 服務器

### 功能
1. **首頁登錄**
   - 房東驗證碼登錄 → host.html
   - 接單員驗證碼登錄 → cleaner.html
   - 管理員直接 → admin.html

2. **房東頁面 (host.html)**
   - 我的發布 - 發布訂單
   - 我的房源 - 添加房源

3. **接單員頁面 (cleaner.html)**
   - 搶單大廳 - 開放訂單
   - 我的歷史 - 已接單/已完成

4. **管理員頁面 (admin.html)**
   - 統計數據
   - 清潔工管理
   - 房東管理
   - 訂單管理

### API 端點
- `/api/stats` - 統計
- `/api/properties` - 房源
- `/api/cleaners` - 接單員
- `/api/hosts` - 房東
- `/api/orders` - 訂單
- `/api/hosts/code/{code}` - 房東驗證碼登錄

### 配色
- 主色: #8A9A5B (sage green)
- 背景: #f5f5f5
