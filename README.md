## 📌 封面

**短租清潔管理系統**

幫助短租房東輕鬆管理房源調度、清潔驗收

適用於 Airbnb、Booking、VRBO 等短租平台

---

## 😰 痛點分析

| 痛點 | 描述 |
|------|------|
| 🔄 高周轉壓力 | 退房後需立即清潔否則空置損失 |
| 📍 房源分散 | 每個房源位置不同難以調度 |
| ⏰ 時間緊迫 | 通常只有 2-4 小時清洁窗口 |
| 👷 人員調度 | 誰有空？誰離得近？ |
| ✅ 品質驗收 | 清潔完如何確認品質？ |
| 📦 物資管理 | 床單、衛生用品庫存 |

---

## ✨ 系統功能

| 功能 | 說明 |
|------|------|
| 🏠 房源管理 | 地址、房型、清潔時間標準化 |
| 📋 智能派單 | 自動分配給可用清潔工 |
| 📍 定位打卡 | 確認清潔工到達現場 |
| 📝 清潔清單 | 標準化檢查項目 |
| 📷 照片驗收 | 清潔完成上傳照片確認 |
| 📊 數據統計 | 工作效率一目了然 |

## 🔄 使用流程

```
1. 房東創建清潔任務
   └─ 選擇房源、輸入退房時間

2. 系統自動派單
   └─ 分配給可用清潔工

3. 清潔工打卡開始
   └─ 到達現場定位確認

4. 完成清潔
   └─ 上傳照片填寫清單

5. 房東驗收
   └─ 確認品質，系統記錄統計
```
