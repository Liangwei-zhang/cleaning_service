# SmartClean 短租清潔調度系統 - 用戶手冊

## 📱 系統概述

SmartClean 是一款專為短租房屋設計的清潔調度管理系統，連房東、清潔師傅和管理員於一體。

**訪問地址：** http://your-server-ip/

---

## 👤 用戶角色

| 角色 | 功能 |
|------|------|
| 房東 | 發布清潔任務、管理房源、驗收清潔 |
| 清潔師傅 | 搶單、開始工作、上傳完工照片 |
| 管理員 | 調度管理、查看所有訂單和用戶 |

---

## 🏠 房東端使用指南

### 1. 登錄/註冊

1. 打開首頁，點擊「房東登錄」
2. 輸入手機號和驗證碼（6位數字）
3. 登錄後自動進入房東後台

### 2. 發布清潔任務

1. 點擊「發布訂單」按鈕
2. 填寫訂單信息：
   - **房源**：選擇已有房源或新增
   - **地址**：詳細地址（會自動解析）
   - **清潔時間**：預約日期和時段
   - **價格**：清潔費用（加幣）
   - **備註**：特殊要求
3. 可選擇**錄音留言**說明要求
4. 點擊「發布」

### 3. 管理房源

1. 點擊「房源管理」
2. 可新增、編輯、刪除房源信息
3. 房源包括：名稱、地址、樓層、面積

### 4. 驗收清潔

1. 收到清潔師傅「完成」通知後
2. 進入「我的訂單」
3. 查看清潔師傅上傳的完工照片
4. 點擊「驗收」確認完成
5. 如有問題可點擊「拒絕」

### 5. 發布訂單（語音）

- 長按話筒圖標開始錄音
- 松開停止錄音
- 可預覽或重新錄製
- 確認後自動保存

## 🧹 清潔師傅端使用指南

### 1. 登錄

1. 點擊「清潔師傅登錄」
2. 輸入手機號和驗證碼
3. 進入搶單大廳

### 2. 搶單

1. 在「搶單大廳」瀏覽待接訂單
2. 可按**價格**或**時間**排序
3. 點擊「接單」搶下訂單
4. 搶單後進入「我的任務」

### 3. 工作流程

| 狀態 | 操作 |
|------|------|
| 已接單 | 開始工作前點「已到達」 |
| 已到達 | 開始清潔，點「開始清潔」 |
| 清潔中 | 完成後點「完成」 |
| 已完成 | 等待房東驗收 |

### 4. 上傳完工照片

1. 清潔完成後
2. 點擊「上傳照片」
3. 最多上傳 5 張照片
4. 系統會自動壓縮

### 5. 播放語音備註

- 在訂單詳情中點擊播放圖標
- 可聽取房東的語音留言

## ⚙️ 管理員功能

### 1. 訪問

管理員登錄後自動進入管理後台

### 2. 功能模塊

- **清潔夥伴**：查看、清潔師傅列表
- **房東**：查看、管理房東
- **訂單**：查看所有訂單
- **統計**：查看運營數據

## 📝 常見問題

### Q1: 忘記手機號？
聯繫管理員處理

### Q2: 接錯訂單怎麼辦？
聯繫房東或管理員取消

### Q3: 照片上傳失敗？
檢查網絡，確保照片小於 5MB

### Q4: 無法驗收？
確保所有照片已上傳
